Explicación de un programa en C

¿Qué hace el programa?

Este programa pregunta la edad de una persona y después dice si es mayor o menor de edad.

Código en C

#include <stdio.h>

int main() {
int edad;

printf("¿Cuántos años tienes?");
scanf("%d", &edad);

if (edad >= 18) {
printf("Eres mayor de edad.");
} else {
printf("Eres menor de edad.");
}

return 0;
}

1. #include <stdio.h>

Esta línea importa una biblioteca llamada stdio.h. Sirve para usar printf y scanf.

2. int main()

Es la función principal del programa. Aquí empieza el código.

3. int edad;

Se crea una variable llamada edad. Una variable sirve para guardar datos. En este caso guardará un número entero.

4. printf()

printf sirve para mostrar mensajes en pantalla.

En este caso pregunta: ¿Cuántos años tienes?

5. scanf("%d", &edad);

scanf sirve para leer datos que escribe el usuario.

El "%d" significa que espera un número entero.

El símbolo "&" ayuda a guardar el dato dentro de la variable edad.

6. if (edad >= 18)

if es una condición.

Revisa si la edad es mayor o igual a 18.

El símbolo >= significa "mayor o igual que".

7. printf("Eres mayor de edad")

Si la condición se cumple, el programa imprime: Eres mayor de edad.

8. else

else significa "si no".

Se usa cuando la condición del if no se cumple.

9. printf("Eres menor de edad")

Si la persona tiene menos de 18 años, muestra: Eres menor de edad.

10. return 0;

Significa que el programa terminó correctamente.
